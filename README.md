## 项目概述

本系统是一个端到端的世界杯预测框架，核心方法论称为**「量化足球」**——将足球比赛的胜负分解为6个层级、13个可量化因子，通过泊松期望进球模型和蒙特卡洛模拟，输出每场比赛的胜/平/负概率、预期比分和冠军概率。

### 核心指标

| 指标 | 数值 |
|------|------|
| 已赛验证 | 28场实际结果对比 |
| 方向准确率 | **16/28 = 57.1%** |
| 比分精确命中 | **8/28 = 28.6%** |
| 蒙特卡洛模拟 | 10,000次赛事仿真 |
| 覆盖范围 | 全部48支球队·104场比赛 |

## 关键创新点

### 1. 首发11人化学引擎（独创）
不同于简单的平均OVR或德转身价，我们计算了每支球队**大概率首发11人**之间的真实化学作用：
```
化学 = 0.22×同俱乐部 + 0.18×合练因子 + 0.18×同联赛 + 0.12×领袖因子 
       + 0.15×位置互补 − 0.15×位置重叠
```
化学分经跨队归一化至[15, 85]区间，数据来源为EA FC 26真实球员评分。

### 2. 大赛基因因子（完全独创）
在任何公开文献和开源项目中**找不到先例**的维度：
- 足球文化指数（巴西98、阿根廷95）
- 政治/社会压力系数（巴西95、阿根廷95）
- 历史大赛超常系数（德国92、摩洛哥85）
- 民族荣誉驱动（阿根廷95、摩洛哥92）

### 3. 场馆感知环境动态注入
2026世界杯16座场馆跨北美三国，海拔2m-2240m，时区UTC-8至-4。系统为每场比赛动态计算：
- 海拔惩罚（-0.06/1000m，杯赛适应版）
- 时差惩罚（3-6h区间-0.010/h，6-9h区间-0.017/h）
- 高温+湿度放大效应

### 4. 三东道主主场校正
仅美国/加拿大/墨西哥三国在本国境内场馆享有β_home=0.12（≈13%进球提升），其余45队均为中立场地——纠正了大多数模型简单套用联赛主场优势的错误。

### 5. 非线性球星依赖惩罚
```
penalty = tanh((star_dependency − 0.12) × 8) × 0.18
```
基于首发中超巨（OPR>82）占比，对过度依赖少数球星的球队施加非线性惩罚。

## 两轮实战调优历程

### 第一阶段（前10场后 → v3）
1. **HOME_ADVANTAGE** 0.35→0.12（东道主优势从42%降至13%）
2. **攻击权重** ×1.10，**防守权重** ×1.20
3. **泊松α** 0.30→0.20（减少平局集中）

### 第二阶段（28场后 → v3.5）
1. **化学权重**：同俱乐部0.30→0.22，合练0.10→0.18
2. **环境惩罚**：海拔-0.08→-0.06/1000m，时差系数下调~30%
3. **层权重**：战术因子15%→18%，大赛基因15%→12%
